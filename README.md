### Project Overview
This project analyzes survey data from data professionals using Power BI to uncover insights into salary trends, job roles, programming preferences, work satisfaction, and industry distribution.
The objective was to clean and transform the raw survey dataset, perform calculations, and design an interactive dashboard that provides meaningful, decision-ready insights.

### Project Objectives
- Analyze salary trends across roles and regions
- Compare job satisfaction levels
- Identify popular programming languages
- Explore industry distribution
- Calculate average compensation metrics
- Create an interactive dashboard for executive-level insights



### Dataset Description
- Source File: Power BI - Final Project.xlsx
- Dashboard File: Data Professionals Survey (Power BI Project).pbix
- Dataset Type: Survey responses from data professionals
- Key Fields Included:
  - Job Title
  - Salary
  - Country
  - Programming Language
  - Work Satisfaction
  - Industry



### Data Preparation
Before building the dashboard, several data transformation steps were performed in Power Query:

1. Data Cleaning

   - Removed unnecessary columns

   - Handled null and blank values

   - Standardized categorical entries


2. Column Simplification

   - Split complex fields into structured columns

   - Reformatted salary and categorical fields
  
     

3.  Calculated Columns & Measures

    - Created average salary measures
    - Calculated aggregated metrics
    - Generated summary statistics
    - Built calculated columns for analysis segmentation

These transformations ensured the dataset was structured for accurate modeling and visualization.



### Dashboard Features
The Power BI dashboard includes:
- Average Salary by Job Role
- Salary Distribution by Country
- Most Popular Programming Languages
- Work-Life Balance & Job Satisfaction Insights
- Industry Breakdown of Respondents
- Interactive filters and slicers for dynamic exploration

The dashboard allows users to filter by role, country, and other attributes to explore patterns interactively.



### Skills and Technologies Used
- Power BI
- Power Query
- DAX
- Microsoft Excel


### Key Insights Identified
- Salary levels vary significantly by job title and region.
- Certain programming languages dominate across roles.
- Job satisfaction correlates with compensation and role specialization.
- Some industries show stronger representation of data roles than others.


### Business Relevance
This dashboard demonstrates how survey data can be transformed into:
- Workforce compensation analysis
- Talent market insights
- Industry benchmarking
- Executive-level reporting tools
- HR strategy support


### Limitations
- Survey data may contain self-reported bias.
- Sample size may not represent the global data market fully.
- Currency standardization assumptions may impact salary comparisons.
